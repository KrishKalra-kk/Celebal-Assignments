# Assignment 2 - Data Engineering

## Overview
SQL analysis on the Sample Superstore dataset using Python and SQLite.

## Dataset
Sample - Superstore.csv with around 10,000 rows of US retail orders from 2014 to 2017.

## Steps Covered
1. Loaded CSV into SQLite database
2. Explored schema and sample data
3. Applied WHERE filters on region, category, date and sales
4. Used GROUP BY for aggregations
5. Sorted and limited results for top products and categories
6. Solved use cases like monthly trends, top customers and duplicates
7. Validated data quality and row counts

## Extra Columns Added
- Profit_Margin_Pct
- Revenue_per_Unit
- Ship_Delay_Days
- Profit_Flag
- Discount_Band

## How to Run
Install pandas and jupyter, then run jupyter notebook and open the ipynb file.

## Tech Used
Python, pandas, SQLite, Jupyter Notebook
