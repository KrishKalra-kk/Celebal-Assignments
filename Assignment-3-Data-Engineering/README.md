# Assignment 3 - Data Engineering

## Overview
SQL analysis using Subqueries, CTEs and Window Functions on the Superstore dataset.

## Dataset
week3sample.csv with around 10,000 rows of US retail orders from 2014 to 2017.

## Steps Covered
1. Loaded data into superstore_raw table
2. Created 3 normalized tables - customers, orders, products
3. Used subqueries to find above average orders and highest order per customer
4. Used CTEs to calculate total sales per customer
5. Applied window functions RANK and ROW_NUMBER for customer ranking
6. Wrote a final combined query using JOIN + CTE + Window Function
7. Solved 5 business questions in the mini project

## Tech Used
Python, pandas, SQLite, Jupyter Notebook
