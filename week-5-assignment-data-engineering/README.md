# Week 5 - Spark Assignment

This is my week 5 assignment for the Data Engineering course. The assignment covers PySpark basics including data cleaning, filtering, and aggregations.

## What's inside

- `Week5_Spark_Assignment.ipynb` - the main notebook with all 15 questions answered
- `week5_dataset.csv` - the dataset used in the notebook

## How to run

You need Python, Java (JDK 11+), and PySpark installed.

```bash
pip install pyspark
```

Then just open the notebook in Jupyter and run all cells from top to bottom. Make sure the CSV file is in the same folder as the notebook.

## Dataset

The dataset has transaction records with columns like user_id, region, product_category, sale_amount, city, age, subscription, price, and store_id. Some columns have null values which are handled in the notebook.

## Questions covered

Q1 to Q15 covering MapReduce vs Spark, in-memory computing, removing duplicates, handling nulls, filtering, groupBy, aggregations, wide transformations, schema changes, and a final data pipeline.
